Push your files from VS Code
Since you're already in VS Code, this is easy. Open the terminal (Ctrl+``  ``) and run:
bashgit init

git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/amon-noritual/tearoom-noritual.git
git push -u origin main

Enable GitHub Pages

Go to your repo on GitHub
Click Settings → Pages (left sidebar)
Under "Source" select Deploy from a branch
Choose main branch, folder / (root)
Click Save

Within ~60 seconds your site will be live at: https://amon-noritual.github.io/tearoom-noritual/

site is live! Now for the QR code — quickest way:

Go to qrcode-monkey.com
Paste your URL: https://amon-noritual.github.io/tearoom-noritual/
Click "Create QR Code"
Click "Download PNG" — choose the highest resolution for printing (at least 1000px)

That's it — print-ready QR code in under a minute.
Tips for printing:

Download as SVG if you plan to print large (it scales infinitely without losing quality)
Make sure the QR code is at least 3×3 cm on the printed page so phones can scan it easily
Leave a small white border around it — scanners need the quiet zone to read it correctly
